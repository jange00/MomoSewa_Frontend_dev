# Test Cases for Momosewa

This directory contains test cases organized by major features.

## Test Case Structure

Each test case follows the standard template format with:
- Project name
- Test case ID
- Module name (Test title)
- Description
- Context
- Pre conditions
- Dependencies
- Step-by-step test steps with Test data, Expected results, Actual results, Status, and Notes
- Post conditions
